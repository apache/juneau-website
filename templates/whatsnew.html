What's New in Juneau 7.2.2

7.2.2 is a minor release consisting of minor bug fixes and improvements to the PetStore application.

What's New in Juneau 7.2.1

7.2.1 is a minor release.

  • Fixed binary backwards compatibility issues with 7.1.0.
  • Method-level annotations (e.g. @RestMethod) and parameter-level annotations (e.g. @Query) are now inheritable from parent classes and interfaces.
    This allows you to define Dual-Purpose Java Interfaces.

Refer to the Release Notes for the full list of changes.

What's New in Juneau 7.2.0

7.2.0 is a major release that introduces several significant new features:

  • OpenAPI part serializing and parsing with full support for OpenAPI validation of input and output in the REST servlet and client APIs.
    • juneau-marshall > OpenAPI Details
    • juneau-rest-server > OpenAPI Schema Part Parsing
    • juneau-rest-server > OpenAPI Schema Part Serializing
  • Auto-generated Swagger UI.
    • juneau-dto > Swagger UI
    • juneau-rest-server > Swagger
  • New HTTP-Part annotations that are applicable to both the servlet and client APIs.
    • juneau-rest-server > HTTP-Part Annotations
    • juneau-rest-client > REST Proxies
  • Simplified UI customization.
    • juneau-microservice-server > UI Customization
  • New Marshall API that combines serializers and parsers into single classes.
    • juneau-marshall > Marshalls
  • Serverless servlet and client unit testing.
    • juneau-rest-server > Serverless Unit Testing
    • juneau-rest-client > Serverless Unit Testing
  • Predefined REST responses and exceptions.
    • juneau-rest-server > Predefined Responses
    • juneau-rest-server > Predefined Exceptions

Refer to the Release Notes for the full list of changes.
