031/**
032 * Serializes POJO models to HTML.
033 *
034 * <h5 class='topic'>Media types</h5>
035 * <p>
036 * Handles <c>Accept</c> types:  <bc>text/html</bc>
037 * <p>
038 * Produces <c>Content-Type</c> types:  <bc>text/html</bc>
039 *
040 * <h5 class='topic'>Description</h5>
041 * <p>
042 * The conversion is as follows...
043 * <ul class='spaced-list'>
044 *    <li>
045 *       {@link Map Maps} (e.g. {@link HashMap}, {@link TreeMap}) and beans are converted to HTML tables with
046 *       'key' and 'value' columns.
047 *    <li>
048 *       {@link Collection Collections} (e.g. {@link HashSet}, {@link LinkedList}) and Java arrays are converted
049 *       to HTML ordered lists.
050 *    <li>
051 *       {@code Collections} of {@code Maps} and beans are converted to HTML tables with keys as headers.
052 *    <li>
053 *       Everything else is converted to text.
054 * </ul>
055 *
056 * <p>
057 * This serializer provides several serialization options.  Typically, one of the predefined <jsf>DEFAULT</jsf>
058 * serializers will be sufficient.
059 * However, custom serializers can be constructed to fine-tune behavior.
060 *
061 * <p>
062 * The {@link HtmlLink} annotation can be used on beans to add hyperlinks to the output.
063 *
064 * <h5 class='topic'>Behavior-specific subclasses</h5>
065 * <p>
066 * The following direct subclasses are provided for convenience:
067 * <ul class='spaced-list'>
068 *    <li>
069 *       {@link Sq} - Default serializer, single quotes.
070 *    <li>
071 *       {@link SqReadable} - Default serializer, single quotes, whitespace added.
072 * </ul>
073 *
074 * <h5 class='section'>Example:</h5>
075 * <p class='bjava'>
076 *    <jc>// Use one of the default serializers to serialize a POJO</jc>
077 *    String <jv>html</jv> = HtmlSerializer.<jsf>DEFAULT</jsf>.serialize(<jv>someObject</jv>);
078 *
079 *    <jc>// Create a custom serializer that doesn't use whitespace and newlines</jc>
080 *    HtmlSerializer <jv>serializer</jv> = HtmlSerializer.<jsm>create</jsm>().ws().build();
081 *
082 *    <jc>// Same as above, except uses cloning</jc>
083 *    HtmlSerializer <jv>serializer</jv> = HtmlSerializer.<jsf>DEFAULT</jsf>.copy().ws().build();
084 *
085 *    <jc>// Serialize POJOs to HTML</jc>
086 *
087 *    <jc>// Produces: </jc>
088 *    <jc>// &lt;ul&gt;&lt;li&gt;1&lt;li&gt;2&lt;li&gt;3&lt;/ul&gt;</jc>
089 *    List <jv>list</jv> = JsonList.<jsm>of</jsm>(1, 2, 3);
090 *    String <jv>html</jv> = HtmlSerializer.<jsf>DEFAULT</jsf>.serialize(<jv>list</jv>);
091 *
092 *    <jc>// Produces: </jc>
093 *    <jc>//    &lt;table&gt; </jc>
094 *    <jc>//       &lt;tr&gt;&lt;th&gt;firstName&lt;/th&gt;&lt;th&gt;lastName&lt;/th&gt;&lt;/tr&gt; </jc>
095 *    <jc>//       &lt;tr&gt;&lt;td&gt;Bob&lt;/td&gt;&lt;td&gt;Costas&lt;/td&gt;&lt;/tr&gt; </jc>
096 *    <jc>//       &lt;tr&gt;&lt;td&gt;Billy&lt;/td&gt;&lt;td&gt;TheKid&lt;/td&gt;&lt;/tr&gt; </jc>
097 *    <jc>//       &lt;tr&gt;&lt;td&gt;Barney&lt;/td&gt;&lt;td&gt;Miller&lt;/td&gt;&lt;/tr&gt; </jc>
098 *    <jc>//    &lt;/table&gt; </jc>
099 *    <jv>html</jv> = JsonList.<jsm>of</jsm>(
100 *       JsonMap.<jsm>ofJson</jsm>(<js>"{firstName:'Bob',lastName:'Costas'}"</js>),
101 *       JsonMap.<jsm>ofJson</jsm>(<js>"{firstName:'Billy',lastName:'TheKid'}"</js>),
102 *       JsonMap.<jsm>ofJson</jsm>(<js>"{firstName:'Barney',lastName:'Miller'}"</js>)
103 *    );
104 *    String <jv>html</jv> = HtmlSerializer.<jsf>DEFAULT</jsf>.serialize(<jv>list</jv>);
105 *
106 *    <jc>// Produces: </jc>
107 *    <jc>//    &lt;table&gt; </jc>
108 *    <jc>//       &lt;tr&gt;&lt;th&gt;key&lt;/th&gt;&lt;th&gt;value&lt;/th&gt;&lt;/tr&gt; </jc>
109 *    <jc>//       &lt;tr&gt;&lt;td&gt;foo&lt;/td&gt;&lt;td&gt;bar&lt;/td&gt;&lt;/tr&gt; </jc>
110 *    <jc>//       &lt;tr&gt;&lt;td&gt;baz&lt;/td&gt;&lt;td&gt;123&lt;/td&gt;&lt;/tr&gt; </jc>
111 *    <jc>//    &lt;/table&gt; </jc>
112 *    Map <jv>map</jv> = JsonMap.<jsm>ofJson</jsm>(<js>"{foo:'bar',baz:123}"</js>);
113 *    String <jv>html</jv> = HtmlSerializer.<jsf>DEFAULT</jsf>.serialize(<jv>map</jv>);
114 *
115 *    <jc>// HTML elements can be nested arbitrarily deep</jc>
116 *    <jc>// Produces: </jc>
117 *    <jc>//   &lt;table&gt; </jc>
118 *    <jc>//      &lt;tr&gt;&lt;th&gt;key&lt;/th&gt;&lt;th&gt;value&lt;/th&gt;&lt;/tr&gt; </jc>
119 *    <jc>//      &lt;tr&gt;&lt;td&gt;foo&lt;/td&gt;&lt;td&gt;bar&lt;/td&gt;&lt;/tr&gt; </jc>
120 *    <jc>//      &lt;tr&gt;&lt;td&gt;baz&lt;/td&gt;&lt;td&gt;123&lt;/td&gt;&lt;/tr&gt; </jc>
121 *    <jc>//      &lt;tr&gt;&lt;td&gt;someNumbers&lt;/td&gt;&lt;td&gt;&lt;ul&gt;&lt;li&gt;1&lt;li&gt;2&lt;li&gt;3&lt;/ul&gt;&lt;/td&gt;&lt;/tr&gt; </jc>
122 *    <jc>//      &lt;tr&gt;&lt;td&gt;someSubMap&lt;/td&gt;&lt;td&gt; </jc>
123 *    <jc>//         &lt;table&gt; </jc>
124 *    <jc>//            &lt;tr&gt;&lt;th&gt;key&lt;/th&gt;&lt;th&gt;value&lt;/th&gt;&lt;/tr&gt; </jc>
125 *    <jc>//            &lt;tr&gt;&lt;td&gt;a&lt;/td&gt;&lt;td&gt;b&lt;/td&gt;&lt;/tr&gt; </jc>
126 *    <jc>//         &lt;/table&gt; </jc>
127 *    <jc>//      &lt;/td&gt;&lt;/tr&gt; </jc>
128 *    <jc>//   &lt;/table&gt; </jc>
129 *    Map <jv>map</jv> = JsonMap.<jsm>ofJson</jsm>(<js>"{foo:'bar',baz:123}"</js>);
130 *    <jv>map</jv>.put(<js>"someNumbers"</js>, JsonList.<jsm>of</jsm>(1, 2, 3));
131 *    <jv>map</jv>.put(<js>"someSubMap"</js>, JsonMap.<jsm>ofJson</jsm>(<js>"{a:'b'}"</js>));
132 *    String <jv>html</jv> = HtmlSerializer.<jsf>DEFAULT</jsf>.serialize(<jv>map</jv>);
133 * </p>
134 *
135 * <ul class='notes'>
136 *    <li class='note'>This class is thread safe and reusable.
137 * </ul>
138 *
139 * <ul class='seealso'>
140 *    <li class='link'>{@doc jm.HtmlDetails}
141 *    <li class='extlink'>{@source}
142 * </ul>
143 */

554      /**
555       * <i><l>HtmlSerializer</l> configuration property:&emsp;</i>  Anchor text source.
556       *
557       * <p>
558       * When creating anchor tags (e.g. <code><xt>&lt;a</xt> <xa>href</xa>=<xs>'...'</xs>
559       * <xt>&gt;</xt>text<xt>&lt;/a&gt;</xt></code>) in HTML, this setting defines what to set the inner text to.
560       *
561       * <p>
562       * The possible values are:
563       * <ul>
564       *    <li class='jc'>{@link AnchorText}
565       *    <ul>
